# Vue 3 + Cli + i18n V9

### Setup

Instalando o `vue-cli`

```
npm install -g @vue/cli
```
Instalando as depêndencias

```
npm install
```

### Compilação e execução local
```
npm serve
```

### Compilação e mimificação para produção

```
npm build
```
