# Vue 3 + Cli + i18n

## Project setup
```
npm install
```
### Adicionando a dependência i18n V9

```
npm install vue-i18n@9
```

### Compiles and hot-reloads for development
```
npm serve
```

### Compiles and minifies for production
```
npm build
```
