# Vue 3 + Vite + i18n V9

### Setup

```
npm install
```

### Adicionando a dependência i18n V9

```
npm install vue-i18n@9
```

### Compilação e execução local
```
npm dev
```

### Compilação e mimificação para produção

```
npm build
```
